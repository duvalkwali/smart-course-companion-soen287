# Installation Guide

1. Download the project folder
2. Open a terminal
3. Navigate into the project folder
```
cd smart-course-companion
```
4. Install the required dependencies:
```
npm install
```
5. Run the React app
```
npm run dev
```
6. Open your browser
7. Visit http://localhost:5173/

# User Guide

1. Create an Account
	1. Click Get Started for Free on the landing page.
	2. Enter the required information.
	3. Click Sign Up either as a student or as an admin.
2. Sign In
	1. Click Sign In in the navigation bar.
	2. Enter the required information.
	3. Click Sign In to access either the student or admin dashboard.
3. Student Dashboard
	1. View your list of courses.
    2. Click View Assessments to see assessments for that course.
    3. Click Assessments in the navigation bar to see upcoming assessments and create new assessments.
    4. Click Add Courses in the navigation bar to add new courses.
    5. Click Sign Out in the navigation bar to sign out and return to landing page
4. Admin Dashboard
    1. View list of all courses and student/course statistics
    2. Toggle courses as enabled or disabled
    3. Click the eye or pencil button to view or edit course details and assessments
    5. Click Add Courses tab to add courses
    6. Click Sign Out in the navigation bar to sign out and return to landing page
